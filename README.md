Blog Central – Spring Boot app 
A full-featured blog built using Spring Boot, MySQL, JPA, JWT Security, and REST API best practices.  
Users can create blogposts, comment, like posts, and manage their profiles.  
All functionality is tested via Postman.

FEATURES:
Public (Visitors)
- View blog posts
- View blog post details
- Search blogposts
- Register & log in

Authenticated Users
- Manage profile (view/update/delete)
- Create blogposts
- Edit/delete own blogposts
- Write comments
- Edit/delete own comments
- Moderate comments on own blogposts
- Like blogposts

Admin
- Full access to all content and moderation
